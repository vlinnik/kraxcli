Команды
=======

Основные команды

- dev connect — установить соединение с контроллером
- dev repl — открыть интерактивную консоль
- fs put — отправить файл(ы) на контроллер
- fs get — скачать файл(ы) с контроллера
- dev reset — перезагрузить контроллер
- dev run — запустить скрипт на контроллере
- dev load — отправить скрипт на контроллер и выполнить (не сохраняя)
